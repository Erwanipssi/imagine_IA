# Imagine AI

Application de génération d'histoires et comptines personnalisées pour enfants (IA locale).

## Lancement rapide avec Docker

```bash
docker compose up -d --build
```

**Accès** : [http://localhost](http://localhost)

> Premier démarrage : 5–10 min (téléchargement images + modèle Ollama ~1.3 Go).

## Arrêter

```bash
docker compose down
```
